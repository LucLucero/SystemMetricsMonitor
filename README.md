## Instalação
Para instalar as dependências necessárias, execute o seguinte comando:

Para executar este projeto, você precisará instalar as seguintes bibliotecas Python:

- **psutil**:
  ```bash
  pip install psutil
- **psycopg2-binary**:
  ```bash
  pip install psycopg2-binary
- **Matplotlib**:
  ```bash
  pip install matplotlib

- **Pandas**:
  ```bash
  pip install pandas
